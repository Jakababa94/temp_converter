# Temperature Converter

A simple, interactive web-based temperature converter that instantly converts between Celsius, Fahrenheit, and Kelvin.

## Features

- **Real-time conversion**: Enter a temperature in any unit and see instant conversions to the other two scales
- **Three temperature scales**: Celsius, Fahrenheit, and Kelvin
- **Clean UI**: Modern card-based interface with glassmorphism design
- **Responsive**: Works on desktop and mobile devices

## How to Use

1. Open `index.html` in a web browser
2. Enter a temperature value in any of the three input fields (Celsius, Fahrenheit, or Kelvin)
3. The corresponding values in the other two scales will automatically update
4. Clear an input field to reset all values

## Files

- `index.html` - Main HTML structure with input fields for each temperature scale
- `script.js` - JavaScript logic for temperature conversions between scales
- `style.css` - Styling with glassmorphism effects and responsive layout

## Conversion Formulas

- **Celsius to Fahrenheit**: °F = (°C × 9/5) + 32
- **Celsius to Kelvin**: K = °C + 273.15
- **Fahrenheit to Celsius**: °C = (°F - 32) × 5/9
- **Kelvin to Celsius**: °C = K - 273.15

## Installation

No installation required! Simply open the `index.html` file in any modern web browser.

## Technologies Used

- HTML5
- CSS3 (with backdrop-filter for glassmorphism)
- Vanilla JavaScript (ES6+)
